# 01 - Documentation And Contract Preservation

## Goals
- Preserve all pydantic contracts.
- Preserve behavior descriptions only (not implementation).
- Use `data/Archived/` as successful reference corpus.

## Tasks
1. Build contract index by node and artifact type.
2. Build behavior description index for each old node.
3. Map archived examples to expected outcomes.
4. Inventory deterministic tools that are reusable as-is.

## Exit Criteria
- Contract index complete.
- Description index complete.
- Archived reference mapping complete.
