# 02 - Hard Deletion

## Goals
- Remove all old runtime implementation except contracts, graph reference, and deterministic tools.

## Rules
- No placeholders.
- No stubs.
- No temporary emulation layer.

## Delete Targets
- Legacy AI runtime modules.
- Legacy prompt execution modules.
- Legacy node logic modules.
- Legacy review parser modules.

## Exit Criteria
- Remaining runtime code is only contracts + graph reference + deterministic tools.
