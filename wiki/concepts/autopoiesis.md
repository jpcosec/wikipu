# Autopoiesis

A system property that allows it to maintain and reproduce itself. The knowledge base uses its own rules and tools to refine its structure.

## Definition

The knowledge base is a self-maintaining system that metabolizes information into coherent topology.

## Examples

- An agent auditing its own docstrings
- Identifying redundant nodes and proposing a merge
