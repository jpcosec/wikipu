### Surfaces

```css
--bg:                  #0c0e10   /* absolute background */
--surface:             #121416   /* base surface */
--surface-low:         #1a1c1e
--surface-container:   #1e2022
--surface-high:        #282a2c
--surface-highest:     #333537
```

### Primary (Cyan)

```css
--primary:             #00f2ff   /* pure accent (glow, active borders) */
--primary-dim:         #99f7ff   /* softer variant */
--on-primary:          #00363a
```

### Secondary (Amber)

```css
--secondary:           #ffaa00   /* deadlines, control panel headers */
--secondary-dim:       #fecb00
--on-secondary:        #452b00
```

### Error


### Text

```css
--text-main:           #e2e2e5   /* on-surface */
```
