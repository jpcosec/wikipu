### Font Families

```css
font-headline: Space Grotesk  /* headers, nav links, panel titles */
font-body:     Inter           /* body text, descriptions */
font-mono:     JetBrains Mono  /* IDs, timestamps, JSON, badges, tags */
```

### Usage Conventions

| Element | Font | Style |
|---------|------|-------|
| App name (`PHD_OS // INTEL_REVIEW`) | headline | bold, uppercase, tracking-tight |
| Section labels | mono | uppercase, tracking-widest, text-muted |
| Body content | body | regular |
| IDs, scores, status | mono | xs/[10px], uppercase |
