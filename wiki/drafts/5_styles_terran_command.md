## Details

All new components follow the existing Terran Command design system:

- **Tab nav:** uppercase Space Grotesk, cyan underline on active tab, ghost style on inactive
- **Plan list:** surface background cards, status badges (draft=muted, active=primary, approved=success, superseded=outline)
- **Iteration timeline:** horizontal chain with dots and connecting lines, plan nodes are cyan, review nodes are amber
- **Touch chips:** compact badges with file icon, cyan border, hover to preview path
- **Comment dialog:** surface modal with mono-font textarea, primary save button
- **File tree badges:** small cyan dot indicator on touched files/folders
- **Tag highlights:** amber background spans (same as IntelligentEditor's cm-highlight-must)
